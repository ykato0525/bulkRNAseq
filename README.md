# bulk RNA-seqのカウント値の算出を自動化する

## 実行環境
WSL2上のUbuntu20.04を想定

## 実行するライブラリ
- FastQC
- multiQC
- fastp
- STAR
- RSEM
- biomart

## 実行環境の構築
pythonのパッケージインストーラーのmambaを利用する


## 手順
パッケージのインストーラーのmambaを使用して、必要なツールのダウンロード
